## 前端技术栈
•  前端：Vue、Vue Router、Axios、Vite、Node.js
•  后端：Java 17、Spring Boot、Spring MVC、Spring Security、JWT、Hibernate Validator
•  数据访问：MyBatis、MyBatis-Plus、Dynamic Datasource
•  数据库：MySQL、AWS RDS
•  缓存/中间件：Redis、Redisson、AWS ElastiCache
•  文件与图片存储：AWS S3、Thumbnailator、对象存储抽象服务
•  部署运维：AWS EC2、Nginx、Linux、Maven、HTTPS、域名解析

## 核心业务
•  商品录入
•  多图上传
•  主图/缩略图管理
•  多仓库 / SKU 唯一性（对应二维码）

## 前端运行方式(node版本16+)

```bash
# 进入项目目录
cd ruoyi-wms-lite-vue

# 安装依赖
npm install --registry=https://registry.npmmirror.com

# 启动服务
npm run dev

# 打包生产
npm run build:prod
# 构建测试环境 yarn build:stage
# 构建生产环境 yarn build:prod
# 前端访问地址 http://localhost:80
```
